# Cloudflare & Browserbase: Pioneering Identity for AI Agents

抓取时间：2026-07-11。

Browserbase 认为复杂 web agent workflows 的关键瓶颈是 Identity：网站无法区分 trusted agent 和 malicious bot。Browserbase 与 Cloudflare 合作支持 Web Bot Auth，让 Browserbase 为客户 agent 提供 cryptographically secure proof of identity。

官方类比：像给 agent 一个 passport。当 agent 到达网站时，网站/anti-bot provider 可检查签名，确认它来自 trusted source。

Browserbase 正在与 Cloudflare、pilot customers 和 anti-bot providers 扩大 AI web access。

判断：Agent Identity 是 Browserbase 相比传统 stealth/proxy/captcha 路线的关键战略差异；如果行业接受，它可以把 Browserbase 从“更好的 headless browser”推向“合法 agent traffic 身份层”。
