Docs 说明 Browserless 覆盖 BrowserQL、BaaS、REST APIs、enterprise deployments、MCP、AI integrations。产品分层：BrowserQL 用于 stealth automation，包含 CAPTCHA solving、fingerprint evasion、human behavior simulation；BaaS 通过 WebSocket 把现有 Puppeteer/Playwright 代码连到 managed cloud browsers；REST APIs 用于 screenshots/PDF/scraping/search/crawl/export；AI integrations 用于 agent frameworks、AI SDKs、no-code automation；还支持 MCP、Docker/self-host、private fleet、queue/concurrency、live debug、token roles。
