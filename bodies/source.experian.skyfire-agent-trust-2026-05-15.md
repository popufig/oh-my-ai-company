# Experian、Akamai 与 Skyfire 推进 Agent Trust 和 KYAPay

Experian 官方公告，2026-05-15。

Experian 宣布 Akamai 加入 Agent Trust 伙伴生态，并称 Experian、Akamai、Skyfire 都是 KYAPay initiative 的 active members。方案把人到 Agent 的绑定、身份/同意/委托权限和交易风险 token，与 Akamai 的 edge security 和行为信号结合。

证据边界：能证明三方公开合作与标准参与，不能证明已有多少商户部署、交易或收入。
