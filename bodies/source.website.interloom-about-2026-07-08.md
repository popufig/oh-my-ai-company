# Interloom About

证据级别：S1 官方；抓取质量：full。

关键信息：
- 官方认为企业每天超过 70% 的工作，如 decisions、escalations、problem-solving，发生在传统系统之外。
- AI agents 进入组织后，最大挑战不是技术，而是组织问题：谁管理 agents、如何进入层级、权限与治理如何处理。
- 他们希望 operational teams 在 IT guardrails 内，用自然语言设计、治理和持续改进 workflow。
- 官方概念：Interloom 是第一套真正的 `navigation system for work`，由 vector embeddings、process traces、adaptive knowledge graph 驱动。
- 判断价值：这页说明 Interloom 的底层论点是“企业 agent 生产化的瓶颈在组织记忆、治理和工作模式捕捉”。
