# Conviction Startup Ideas

采集时间：2026-07-08，证据等级 S1。

## 核心信息

这是 Conviction 对 AI startup opportunities 的系统地图。页面不是普通博客，而是展示它如何看 AI 产品机会和市场缺口。

本次特别相关的方向：

- AI-First Tooling：给模型使用的工具不该只是人类工具的复制品，要为 controllability、parallelism、latency 优化。
- Web Data APIs：模型需要实时、可靠、可引用的网页内容；当前 web content APIs 不足以支撑大规模 agent / browsing 应用。
- Ownership is the New Sales：在碎片化服务行业里，与其向低技术采用率客户卖 AI 工具，不如 own and improve these businesses top-down。
- Enterprise Vibe Coding / The Year of Cheap, Bespoke Code：AI coding 让定制化小代码/内部工具变得便宜。
- Always Pick Up the Phone：SMB 来电和 voice agent 场景。
- Autonomous HR / IT Helpdesk：企业内部流程和 helpdesk 自动化。
- Portals as a Service：用 computer-use agent 包装老旧门户，给开发者一个 clean API。

## 对我们的意义

这张 ideas map 和我们最近研究高度重合：browser / web data、enterprise agents、AI coding、SMB voice、AI-enabled services ownership。它可以作为后续选公司和建 concept 的种子库。
