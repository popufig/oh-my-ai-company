# Tasklet 风险与疑问

主要风险：

1. 供应商即竞争对手：EP2 中 Andrew 明确说 Anthropic 既是关键供应商也是最大竞争对手之一，且流失用户中大量去了 Anthropic 产品。Claude Max 与 API 成本差会压缩 Tasklet 毛利和模型选择。
2. 复杂任务可靠性与成本：Reddit 用户既有强正反馈，也有“复杂任务越做越贵、修一个坏另一个、消耗很多 credits”的负反馈。
3. 安全/权限信任：Reddit 用户直接问敏感邮件、文件、数据如何防泄漏。官网有 security/enterprise 文案，但仍需核实 SOC2/GDPR 进展。
4. 收入口径：$5M ARR/run-rate、$7M run-rate、Sacra $10M ARR、YC Jobs $29.5M funding 都是不同口径。报告必须注明来源。
5. 流量-估值张力：Similarweb 估算流量规模不大且 paid search 比重高，说明收入可能依赖非公开渠道、极高转化或 run-rate 口径。
6. Product Hunt 抓取失败：PH 页面存在，但当前只能抓到安全验证页，不能引用评论/排名/票数。

后续应补：完整 HN thread、Product Hunt launch 数据、LinkedIn 员工/公司页、最新 Similarweb、更多 Reddit/X 用户反馈。
