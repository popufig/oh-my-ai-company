# Tasklet sandbox VM

URL: https://tasklet.ai/release-notes/2025-12-16-sandbox-vm
Legacy local source: `vertical/media/knowledge/radar/tasklet/sources/release-notes.cn.md`.

Evidence level: S1 official release note. Quality: full via legacy capture.

Key points:
- Every Tasklet agent gets its own filesystem and VM sandbox, a Linux environment for terminal commands and code execution.
- Tasklet frames this as moving from "here is how you could do it" to "here, I did it".
- Enables file transfer, conversion, creation, editing and analysis workflows.
- Combined with triggers, it supports recurring data/report pipelines.
