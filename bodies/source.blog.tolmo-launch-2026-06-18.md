# Hello. Tolmo. $22M to build the Agent Fleet Securing Production

Tolmo 于 2026-06-18 出隐身并宣布 $22M，由 Accel 领投，Y Combinator 参与。官方称首批 40 家 design partners 已把 Pentesting Agent 部署到生产，并发现数百个 critical findings；Discovery 与 Remediation Agent 正在陆续推出。

文章介绍四位联合创始人 Pierre Betouin、Jean-Baptiste Aviat、Vladimir de Turckheim、Arnaud Breton，并回溯 Sqreen、Datadog、Apple、Snyk 等团队背景。

文章列出多位个人 backer，包括 Tenable、Datadog、Cloudflare、Awake Security 等公司的创始人/高管。这里只作为网络证据，不自动创建所有 angel 节点。

证据边界：客户数、findings 和生产部署均为公司自述，没有具名案例或独立审计。
