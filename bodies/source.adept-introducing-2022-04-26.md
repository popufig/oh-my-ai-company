# Introducing Adept

Source: https://www.adept.ai/blog/introducing-adept/
Published: 2022-04-26
Evidence: S1 official / quality full

Adept’s initial thesis: build models that can use every software tool and API in the world, so people can get things done in front of a computer through a universal collaborator.

Important framing:

- Text models can read/write but cannot take actions in the digital world.
- Adept wanted an overlay within the computer that works hand-in-hand with users using the same tools they do.
- Example tasks: generate monthly compliance report, draw stairs between points in a blueprint, using existing software like Airtable, Photoshop, ATS, Tableau, Twilio.
- Product is framed as a practical and safer path to general intelligence because it is an interface to existing tools with humans in the driver’s seat.
- Announced $65M from Greylock, Addition, Root Ventures and angels.
- The post explicitly names [[person.ashish-vaswani]] and [[person.niki-parmar]] as David Luan’s cofounders and Transformer inventors.
