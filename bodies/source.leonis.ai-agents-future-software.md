# Leonis：AI Agents and the Future of Software

采集时间：2026-07-08。browser read 可抓。

文章核心观点：AI agents 是能在现实世界自主行动的系统，不同于传统 chatbots/copilots。Leonis 认为 agents 将驱动下一代软件，即 Software 3.0。

文章提出的研究方向：

- Agent frameworks：分解任务、加入 ReAct/Reflection、短期/长期记忆、工具集成，但真实部署需要稳定性和性能优化。
- Multi-agent systems：多个专业 agent 协作，可能优于单一 generalist model。
- Strategic reasoning：长程目标、动态环境、self-reflection。

四个预测：

1. Specialized tools will empower vertical agents。
2. Multi-agent systems will outperform single-agent systems。
3. AI agent frameworks will become the new developer tools。
4. AI agents will usher in Software 3.0。

研究价值：这篇解释了 Leonis 为什么会投 [[company.viktor]]，也解释了它 portfolio 中 agent infra、vertical agents、agent security/observability 等公司的来源。
