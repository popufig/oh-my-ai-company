# Wenfeng Wang / 王文锋

Helio 与 Sheet0 创始人，1994 年生，分布式系统、数据平台和基础软件背景。2019 年曾在地平线从事数据平台工作，2021 年参与基础软件创业，2023 年开始围绕 Agent 连续探索。[[source.helio.tencent-founder-interview]] [[source.helio.zpotentials-sheet0-founder]]

他的路线不是简单追逐应用热点，而是一条连续的技术判断：

1. AI Coding：尝试从 PRD 直接生成软件；
2. NPi / Tool Use：尝试自然语言调用工具，后来认识到缺少 Agent 上下文；
3. Sheet0：把实时数据获取和结构化交付作为 Agent 的新后端；
4. Helio：把上下文扩展到组织、频道、任务、日历、邮箱、身份与审批，建立 AI Workforce 容器。

这条路径的优势是对 context、tool use 和长任务基础设施有连续积累；风险是方向多次变化，商业闭环仍需由付费和留存证明。
