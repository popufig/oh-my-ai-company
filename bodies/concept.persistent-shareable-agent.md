# 持久、可共享的个人 Agent

这个概念来自 [[company.hellominds]] 的产品表达：Mind 不是一次性的聊天窗口，而是有身份、长期记忆、工具、技能，并且可以被分享给他人的持续工作实体。

关键结构：

- 持久身份：用户为 agent 设定 identity/goals/personality。
- 长期记忆：跨会话保留 context、decision、conversation。
- 工具与技能：不只是回答，还能接入 Gmail、Calendar、Slack、Dune、Perplexity、SerpAPI 等工具。
- 可共享：一个 Mind 可以被朋友、家人、同事一起使用。
- 后台执行：处理 follow-up、workflow、monitoring，不要求用户持续在线。

研究价值：这可以作为我们观察 personal/prosumer agent 的一个产品模式。后续看同类产品时，可以问：它是否真的持久？是否可共享？是否有权限和工具闭环？是否从高频模板切入？
