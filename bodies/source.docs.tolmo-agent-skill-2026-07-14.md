# Tolmo Agent Skill 文档

`tolmo skill install` 支持把 Tolmo Skill 写入 Claude Code 与通用 `~/.agents` 目录。Skill 给 Agent 开放完整 CLI 能力，包括基础设施查询、finding 管理和 connected service proxy。

文档还描述可选 Claude Code OpenTelemetry，能够采集 tool calls 与 assistant messages。这个接口说明 Tolmo 不只是安全 Dashboard，也在做 Agent 的上下文、工具和审计入口。

原文：https://docs.tolmo.com/guides/agent-skill.md
