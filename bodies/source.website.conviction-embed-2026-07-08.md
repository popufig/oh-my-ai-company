# Conviction Embed Program

首次采集：2026-07-08；更新：2026-07-21。证据等级 S1。

## 核心信息

Embed 是 Conviction 面向早期 AI founding teams 的高度选择性项目。

官方页面信息：

- 目标：帮助 AI-native / Software 3.0 公司。
- 官网当前开放第七期申请；已合作公司包括 Chai、Cognition、Listen Labs、Physical Intelligence、Pika Labs、Reflection、Somite、Yutori。
- 当前条款：$250K investment，uncapped no-discount MFN SAFE，或按最近 60 天 round 价格。
- 资源：$350K AWS credits、$350K Azure credits、$500K+ other compute / inference / hosting credits，包括 OpenAI、Anthropic、Baseten、Pinecone、Vercel、Weights & Biases。
- 早期 Mistral models access。
- 每期 10-12 家公司，前几批选择率小于 1%。
- Advisors / visiting partners 包括 Notion、Brex、Atlassian、Stripe、Replit、Shopify、LangChain、Retool、Vercel、Figma、Mistral、Andrej Karpathy 等。

## 对 Conviction 的意义

Embed 是 Conviction 的“早期 AI 公司雷达 + 社区 + 人才/算力/融资平台”。它让 Conviction 不只是被动等 pitch，而是主动组织 AI 创业者、人才和客户网络。

## 边界

Conviction 官网首页把 Embed 称为 grant program，但申请页与 FAQ 明确写的是 SAFE investment；研究中应保留后者的法律/经济条款，不把“grant”理解为无股权资助。Credits 是名义额度，不代表每个团队都会全部使用，也不能直接换算为产品优势或现金价值。
