# linux.do: Web Agent 岗位招聘中的 Browserbase/Stagehand 技能信号

证据等级：S3（中文招聘/社区帖子）

这是一条 Web Agent 方向高级工程师招聘帖，虽然不是 Browserbase 自身信息，但它把 Browserbase、Stagehand、browser-use、Skyvern 列为加分经验，能作为市场技能需求的弱信号。

岗位描述强调：浏览器运行时、session 隔离、profile 管理、截图录屏、live view、登录态持久化、崩溃恢复、长程任务状态机、失败归因、反爬策略调整、CAPTCHA/强验证识别与上报、结构化结果归档、SSE/SDK/错误码、可观测性。

判断：Browserbase 所在能力已经从“网页自动化工具”扩展成 Web Agent runtime 工程岗位要求。对 cp 的产品判断更重要的是这组工程清单：运行时、长程任务、状态恢复、可观测、HITL、登录态和失败分类。
