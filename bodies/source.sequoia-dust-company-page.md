# Sequoia portfolio page: Dust

Sequoia portfolio page 把 Dust 描述为 “AI operating system for smart teams”，强调多模型处理 workplace information，让信息在需要时以合适格式可用。

页面显示：Founded 2023，Partnered 2023。团队列出 [[person.gabriel-hubert]] 和 [[person.stanislas-polu]]。Sequoia partners 包括 Konstantine Buhler 和 Julien Bek。

研究价值：Sequoia 对 Dust 的定位不是简单 chatbot，而是 workplace information / productivity / operating system。这个口径和 Dust 自己的 multiplayer AI workspace 叙事一致。
