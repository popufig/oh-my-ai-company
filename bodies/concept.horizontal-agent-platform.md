# 横向 Agent 工作平台

一句话：不是给某个垂直场景嵌入 AI，而是用一个通用 agent 平台承接知识工作中的工具连接、上下文、执行、UI 和后台自动化。

[[company.tasklet]] 是当前样本：官网和 YC profile 都把它定位成 cloud agent OS / work operating system。Andrew Lee 在 [[source.podcast.tasklet-cogrev-ep2]] 里进一步提出，AI 转型后能活下来的软件形态之一是少数横向平台，Tasklet 想成为知识工作的那个。

这个 concept 用来识别那些试图从“单点工具/工作流自动化”升级为“用户工作入口”的 agent 产品。关键问题不是它能不能做一个任务，而是它能否成为用户维护上下文、连接和权限的长期工作层。
