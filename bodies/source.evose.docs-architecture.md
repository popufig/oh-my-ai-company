官方三层架构：Layer 1 Workbench 供普通员工使用已发布的 Agent、Chatflow、Workflow 和模型，并提供 Library 与 Schedule；Layer 2 Workspace 供产品、开发和业务运营创建、编排、发布并治理应用、知识库、数据源、工具与技能；Layer 3 Organization 供 IT/安全管理成员部门、模型、凭据、API Key、组织工具、观测、资源策略与桌面审计。

一句话边界是：组织层决定“有什么”，工作空间决定“怎么组装”，工作台决定“用户怎么用”。SaaS 为多租户组织→工作空间，私有化为一套部署一个组织且无平台 Credits 计费。
