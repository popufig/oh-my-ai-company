# Hyperagent case study

官方客户案例，已在 [[company.hyperagent]] 调研中使用。抓取时间：2026-07-11。

Browserbase 称 Hyperagent 自 4 月以来在 Browserbase 上跑了 300,000+ sessions。Stagehand powers every web action inside the agent run loop。Live browser view 是每次 agent run 的核心 UX。

Browserbase 为 Hyperagent 提供 browser access、auth walls、global browser infra、session replay、logs、live view。Persistent auth 和 skills library 是后续方向。

判断：Hyperagent case 证明 Browserbase 是 agent fleet / enterprise agent 平台的底层 execution infra，而不是只做 scraper/test automation。
