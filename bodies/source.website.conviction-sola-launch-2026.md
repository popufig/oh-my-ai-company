# Investing in Sola

[[person.pranav-reddy-conviction]] 与 [[person.sarah-guo]] 代表 [[investor.conviction]] 署名的官方投资文章，公开欢迎 a16z 成为 [[company.sola]] 新投资人，并称 Conviction 与 YC 已长期支持团队。

它能证明 Sarah + Pranav 的公开归因、Conviction 的 enterprise automation thesis 与 Sola portfolio 关系。客户使用与节省工时来自投资方文章，未提供客户名、cohort 或独立验证；页面未披露 Conviction 的具体轮次和金额。
