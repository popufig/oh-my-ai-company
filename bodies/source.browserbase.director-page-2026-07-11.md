# Director product page

抓取时间：2026-07-11。正文来自 browser snapshot。

核心定位：Describe it. Watch it happen.

Director 是一个可观察、可纠正、可构建的 browser agent。它把自然语言指令变成 working browser agents；不需要从代码开始，但成功后可以导出真实 Stagehand code。

典型 use cases：

- 每天从 municipal portals 拉新 filings。
- 监控公开页面变化并告警。
- 从 CSV 批量填写 surveys/forms。
- 跨 registry 验证 licenses / credentials / compliance。
- 在没有 bulk export 的 CRM 之间迁移数据。

关键能力：run once、schedule、trigger、scale across thousands of sessions。敏感凭据支持 1Password integration，把 credentials 放在 agent 视野之外。

页面声称：20,000+ developers have run over 50 million sessions on it；客户包括 Vercel、Perplexity、11x。
