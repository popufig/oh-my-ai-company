# Tolmo 三类安全 Agent 产品页

综合读取：

- https://tolmo.com/product/pentesting-agent/
- https://tolmo.com/product/internal-discovery-agent/
- https://tolmo.com/product/remediation-agent/

Pentesting 强调 inside-out、IAM/CI/secrets/trust edges 和 exploitability proof；Discovery 通过 env vars、event mapping、secret refs、IaC 推断关系，并保留节点/边时间历史；Remediation 展示 before/after graph delta 和 blast radius，再把上下文交给 coding agent。

这是官方功能说明，不等于三项能力都已在所有客户可用。融资公告只明确 Pentesting Agent 已生产部署。
