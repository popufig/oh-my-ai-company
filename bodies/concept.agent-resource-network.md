# Agent 资源交易网络

一种 agent infra 产品模式：不是只提供一个工具或模型代理，而是把模型、API、数据、SaaS 工具、技能、运行时和计费结算统一成 agent 可发现、可调用、可付费的资源网络。

AIsa 是当前样本：官网、融资稿、Forbes 和 agent-card 都显示它试图做 agent economy 的 capability / transaction layer。

关键观察：这个模式是否成立，不取决于官网叙事，而取决于 agent 是否真的成为高频资源消费者，以及资源供给侧是否愿意通过统一网络做发现、调用和分成。
