# Dust security page

采集时间：2026-07-09。

Security page 标题：Dust Security: Enterprise-Grade Data Protection。

关键安全口径：

- GDPR compliant。
- SOC 2 Type II certified。
- Enables HIPAA compliance。
- Granular data selection：控制每个 source ingest 哪些数据。
- Encryption：AES-256 at rest，TLS in transit。
- Regional hosting：EU 或 US。
- No model training：客户数据不用于训练模型。
- Zero data retention：third-party model providers 不存储数据。
- Flexible providers：客户可选择 providers 和 embedding models。
- SSO：管理 workspace access。
- Role-based access：user / builder / admin roles。
- Private Spaces：用于敏感数据，按 role 限制访问。

研究判断：这页验证 Dust 在做 enterprise agent 必须面对的数据边界和权限治理。它不是纯 SMB AI 工具，而是明确往企业部署、安全审计和 regulated data 场景走。
