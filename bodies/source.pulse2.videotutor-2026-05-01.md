# Pulse2: VideoTutor seed-backed AI tutoring platform surpasses 50M TikTok views

Pulse2 报道，作者 Amit Chowdhry，2026-05-01。

关键信息：
- $11M seed backing from YZi Labs, Baidu Ventures, Amino Capital。
- VideoTutor-related content surpassed 50M cumulative TikTok views。
- 产品生成 on-demand instructional videos with animated graphs、geometric constructions、formula breakdowns、real-time voice interaction。
- draws on Manim，并自动化从教学结构到视觉组件再到实时 narrated lesson 的流程。
- 学生可以中途打断、追问、要求 alternative explanations。
- 公司披露收到 Tencent、Xiaotiancai 等 interest 和 1000+ API integration requests。
