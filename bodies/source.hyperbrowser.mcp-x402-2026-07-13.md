MCP server 暴露 scrape_webpage、extract_structured_data、crawl_webpages 等工具，并支持 sessionOptions: useStealth、useProxy、solveCaptchas、acceptCookies。安装方式包括 npx hyperbrowser-mcp 和 HYPERBROWSER_API_KEY。

X402 文档显示 Fetch/Search 支持 HTTP 402/USDC 付款，让 agent 可按请求付费访问 web API。这个方向更实验性，但体现它想成为 agent 可调用 web service layer。
