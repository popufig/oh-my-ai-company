# Conviction takeaway：AI VC 的价值从资本变成身份和基础设施

Conviction 这次最值得学的是：在 AI 时代，早期 VC 的价值不只是钱，而是“身份 + 基础设施”。

## 我的判断

对 HeyGen 来说，Conviction 的 $5.6M 不是最关键的；关键是 Sarah Guo 进 board、HongShan board seat 退出，以及 HeyGen 得到一个美国 AI-native investor 的公开背书。

这解决的是美国企业客户、美国资本市场和地缘政治语境下的 trust 问题。

## Conviction 的平台能力

Conviction 做了几件很适合 AI startup 的事：

- 用 No Priors / Sarah Guo 个人品牌获得 AI 圈内容影响力；
- 用 Embed 提前吸引早期 AI founder；
- 用 compute / API credits 降低 AI startup 实验成本；
- 用 talent team、jobs、Mixture of Experts 帮公司招人；
- 用 Startup Ideas 公开定义机会地图，吸引同频 founder；
- 用 portfolio 的成功案例形成飞轮：Harvey、Mistral、Sierra、Cognition、HeyGen。

## 对我们后续研究的影响

以后看到一家 AI 公司有 Conviction，应该提高关注优先级。它可能代表：

- founder 质量较高；
- 方向符合 AI-native / Software 3.0 thesis；
- 公司可能能接入美国 AI founder / researcher / customer 网络；
- 如果公司有跨境背景，Conviction 可能还承担“身份重构/美国市场可信度”的作用。
