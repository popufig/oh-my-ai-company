# Sierra Trust and reliability

S1 官方 trust/reliability 页。Sierra 声称合规覆盖 SOC 2、HIPAA、GDPR、PCI、FedRAMP High、CCPA、CSA STAR、ISO 27001、ISO 42001。

页面强调 deterministic integrations、围绕 LLM 的 supervisory layers、PII 加密/脱敏、PCI DSS Level 1，并将 cardholder data 与 Sierra core platform、LLMs 和 persistent storage 隔离。还提到使用 frontier/open-weight/proprietary model constellation，可自动切换 provider。
