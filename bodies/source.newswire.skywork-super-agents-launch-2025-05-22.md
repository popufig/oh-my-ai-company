# Skywork Super Agents 全球发布 PR

公司 PR。2025-05-22，Skywork 宣布 Skywork Super Agents 面向全球开放，无需邀请码。

核心内容：AI agent architecture + deep research，用于生成 documents、slides、Excel sheets、webpages、podcasts 和 multimodal audiovisual content。公司将其定义为 “AI Office Agent”。

PR 还声明：Skywork deep research agent framework 在 GAIA benchmark 得分 82.42，截至 2025-05-10 排名全球第一，超过 OpenAI Deep Research 和 Manus；SimpleQA 得分 94.5。这些属于公司声明，需第三方验证。

产品结构：5 个专家 agents（Docs、Slides、Sheets、Podcasts、Webpages）+ 1 个 general agent。设计点包括 Clarification Card、traceable sources、Personal Knowledge Base。
