CLOUD.md 对 Browser Use Cloud 的概念解释很完整：核心产品是 task completion。核心对象包括 Session、Browser、Agent、Model、Browser Profile、Task、Profile Sync。

重要细节：Session currently limited to 15 minutes for free users；paid subscribers up to 4 hours。Browser 是 cloud hosted browser，可通过 CDP URL 控制。Browser Profile 是云端 Chrome Profile，用于持久化 cookies/localStorage/saved passwords。Profile Sync 可上传本地 cookies 到 cloud profile。

CLOUD.md 里 browser session pricing 为 $0.05/hour；官网首页主打 stealth browser $0.02/hour。价格口径存在页面/版本差异，应按当前页面和时间标注。
