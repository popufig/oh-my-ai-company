# NanoCorp CLI quickstart
官方 CLI 支持 `npm i -g nanocorp`、`nanocorp login`、`nanocorp init --name acme --one-liner "AI for vets"`。创建后的 company 初始为 paused；用户可运行 `nanocorp skills install`，让 Claude Code 或 Codex 通过 CLI 构建和操作公司。

这让 NanoCorp 从纯 no-code UI 扩张到 coding-agent control surface：公司 runtime 仍由 NanoCorp 托管，但外部 coding agent 可以成为前台操作者。
