Key claims:
- Tasklet rebuilt almost everything in the prior six months: product structure, computer use, integrations, agent core, context management and compaction.
- Files are the agent: history moves into a file system; prompts contain hints/pointers; older context is compressed with decreasing fidelity and cache-aware buckets.
- Computer use became core: headless VM + browser VM with persistent state; if computer use fails, product fails.
- Anthropic is both critical supplier and major competitor; Andrew says 80% of churned users went to Anthropic products and Claude Max provides far more tokens than API at same price.
- Tasklet’s strategic response: become a model-neutral horizontal platform.
- Three software survivors: horizontal platforms, headless/API-first companies, and solution companies selling outcomes.
- Safety/oversight: approval for sensitive tools, rollback/versioning, logs, generating tested migration scripts instead of directly letting LLM transform data.
- Vendors mentioned: Blacksalt for sandbox cold starts, Firecrawl for crawling.
