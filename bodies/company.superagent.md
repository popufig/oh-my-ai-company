# Superagent

Superagent 是 Airtable 在 2026-01-27 发布的 standalone multi-agent research / deliverable 产品。Airtable 官方描述：用户提出复杂问题后，coordinating agent 规划工作，部署 specialist agents 并行调查，最后合成为可直接使用的 interactive deliverable。

它和 [[company.hyperagent]] 的关系：Superagent 更偏“复杂研究问题 -> boardroom-ready answer”；Hyperagent 更偏“组织里的 agent fleet / skills / memory / tool integrations / deployment”。两者都属于 Airtable 的 AI-native multi-product strategy。

证据：[[source.airtable.superagent-launch-2026-01-27]]、[[source.linkedin.superagent-company-2026-07-10]]。
