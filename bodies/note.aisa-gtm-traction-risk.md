# AIsa 增长与 traction 风险

AIsa 当前还不能按“已经跑出平台飞轮”来理解。旧 Similarweb 快照显示 aisa.one 仍是千级/月访问，Direct 占比很高，搜索、referral、paid 都弱。V2EX 和 LinkedIn/GitHub/OpenClaw 更像早期 developer ecosystem seed，而不是成熟增长引擎。

官方 traction 数字很漂亮，但需要拆：50,000+ registered agents、150x registered agent growth、200x API calls/transactions growth 都没有定义口径；Forbes 又写 20,000+ registered agents。下一轮调研要追：注册去重、活跃 agent、真实调用、付费 GMV、资源供给数、生产客户。

GTM 上值得盯：V2EX 招聘显示他们在补 Growth Engineer 和 Developer Advocate，说明团队意识到开发者社区/内容/SEO/KOL/样例项目是关键，不只是融资 PR。
