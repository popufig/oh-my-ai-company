# Accel

Accel 是 Viktor 2026 年 $75M Series A 的领投方。Accel 官方文章 [[source.accel.viktor-investment-2026-05-19]] 把 Viktor 放在 AI adoption 第三波：chat interface -> coding assistants -> omnipresent workplace agents。

Accel 对 Viktor 的核心判断：

- 它不是个人 AI helper，而是 team product。
- 它生活在 Slack / Microsoft Teams 这种团队协作入口里。
- 它连接 3,000+ workplace tools，客户平均连接 30+ tools / data sources。
- 它的价值来自 context + collaboration + cross-system execution。

Accel portfolio page [[source.accel.viktor-company-page-2026-07-08]] 确认：Initial Investment = Series A in 2026，Partner = [[person.zhenya-loginov]]。Zhenya 的背景是 enterprise SaaS / GTM operator：Miro CRO、Segment COO、Dropbox 多职能团队。

研究判断：Accel 这笔不是单纯追热点。它押的是“AI 从个人聊天/编程进入企业协作入口”的迁移，且由有 SaaS/GTM 背景的 partner 主导。
