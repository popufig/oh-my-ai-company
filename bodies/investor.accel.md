# Accel

Accel 是一家全球风险投资机构。当前 vault 已通过独立 `investment` 对象确认其与 [[company.kernel]]、[[company.viktor]]、[[company.lovable]]、[[company.synthflow-ai]]、[[company.solda-ai]] 的投资关系；每条关系的轮次、角色与证据分别保留，不把 portfolio 名单直接当作投资事实。

## Kernel：从浏览器基础设施进入可信 Agent Web

Accel 在 2025-10-09 领投 [[company.kernel]] 的 Seed + Series A 融资。两轮合计 **$22M**，但公开材料没有拆分各轮金额，因此记录为组合融资，不推算单轮规模。结构化关系见 [[investment.accel-kernel-2025]]，证据见 [[source.blog.kernel-series-a-2025-10-09]]、[[source.accel.kernel-investment-2025-10-09]]。

Accel 的投资文章重点不是泛泛的 browser automation，而是三项更接近基础设施护城河的能力：毫秒级浏览器启动、live view/replay 可观察性，以及 agent authentication。它还把 [[person.catherine-jue]] 在 Cash App 遇到的跨商户网站 QA 问题，与 [[person.rafael-garcia]] 在 Clever 的身份和安全经验连接起来。

这条投资线说明 Accel 看中的不只是“给 agent 一台浏览器”，而是浏览器执行、认证和可观察性共同构成的生产运行层。Kernel 后续提出的可信 agent 身份方向进一步强化了这条逻辑，但“first auth platform”仍只是投资方口径，不作为市场第一结论。

## Viktor：从企业协作入口进入 AI 员工

Accel 是 [[company.viktor]] 2026 年 $75M Series A 的领投方。Accel 官方文章 [[source.accel.viktor-investment-2026-05-19]] 把 Viktor 放在 AI adoption 第三波：chat interface -> coding assistants -> omnipresent workplace agents。

Accel 对 Viktor 的核心判断：

- 它不是个人 AI helper，而是 team product。
- 它生活在 Slack / Microsoft Teams 这种团队协作入口里。
- 它连接 3,000+ workplace tools，客户平均连接 30+ tools / data sources。
- 它的价值来自 context + collaboration + cross-system execution。

Accel portfolio page [[source.accel.viktor-company-page-2026-07-08]] 确认：Initial Investment = Series A in 2026，Partner = [[person.zhenya-loginov]]。Zhenya 的背景是 enterprise SaaS / GTM operator：Miro CRO、Segment COO、Dropbox 多职能团队。

研究判断：Accel 这笔不是单纯追热点。它押的是“AI 从个人聊天/编程进入企业协作入口”的迁移，且由有 SaaS/GTM 背景的 partner 主导。

关联：[[company.viktor]]、[[investment.accel-viktor-series-a-2026]]、[[person.zhenya-loginov]]。

围绕 [[person.zhenya-loginov]] 挖出的相邻标的：[[company.lovable]]、[[company.synthflow-ai]]、[[company.solda-ai]]。这些分别代表 AI web app builder、AI voice agent builder、vertical autonomous sales agents，与 [[company.viktor]] 共同指向 AI 从 chat 进入生产系统和企业工作流。
