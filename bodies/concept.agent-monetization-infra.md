# Agent monetization infrastructure

AI agent 从 seat-based SaaS 进入 usage / outcome / performance pricing 后，需要新的 monetization infrastructure：metering、cost tracking、pricing experiments、ROI proof、billing、revenue recognition、customer-facing value proof。

代表样本：[[company.paid]]。Lightspeed 先写 billing thesis，再领投 Paid，把它定义成 AI agent economy 的 business engine。

这和 [[concept.agent-native-payments]] 相邻但不相同：
- agent-native payments 更关注 agent 如何为外部资源付费；
- agent monetization infra 更关注 agent 公司如何向客户收费、证明价值、管理成本和收入。

Evidence: [[source.lsvp.billing-ai-agents-2025-03-06]], [[source.lsvp.paid-investment-2025-09-29]], [[source.portfolio.lightspeed-paid]].
