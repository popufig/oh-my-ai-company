# VideoTutor surpasses 50 million TikTok views

GlobeNewswire 发布，2026-05-01，byline: Videotutor。注意页面声明 sponsored content / reflects views of content provider，因此按 S2/S3 之间处理，不能当独立媒体核验。

关键信息：
- VideoTutor-related content surpassed 50M cumulative TikTok views, driven largely by organic student sharing。
- 公司称 backed by $11M seed from YZi Labs, Baidu Ventures, Amino Capital。
- 产品不同于 text-based answer tools，生成 on-demand instructional videos，包含 animated graphs、geometric constructions、formula breakdowns、real-time voice interaction。
- 技术：draws on Manim；automates question -> teaching explanation -> visual components -> narrated video lesson。
- 用户可 interrupt lessons mid-stream、ask follow-up questions、request alternative explanations。
- 公司称收到 Tencent Group、Xiaotiancai 等 purchase/deployment inquiries，以及 1000+ organizations API integration requests；这是公司披露，需独立验证。
- Quote 来自 Kai Zhao：重点是 developing the explanation itself, not just the answer。
