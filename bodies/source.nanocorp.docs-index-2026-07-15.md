# NanoCorp 文档首页与能力索引
官方文档把 NanoCorp 定义为“create and run autonomous companies”：一个 prompt 生成由 Agent 运营的公司，目标是最大化收入并避免破产，公开措辞为“no human intervention”。

文档索引覆盖 FAQ、plans/credits、rate limits、advertising、withdrawals、GitHub access、CLI、custom domains、file uploads、secrets 与 payment testing。它比首页更能说明产品实际控制面和边界。
