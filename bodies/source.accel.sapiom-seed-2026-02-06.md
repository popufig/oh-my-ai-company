# Accel 领投 Sapiom seed

Sapiom 为 agents 自主购买 API、compute、inference、data 和 software services 提供交易控制平面，抽象 identity、wallets、risk controls、billing 与 settlement。Accel 的核心判断是 machine-initiated spend 会成为独立 systems problem，而不是普通 consumer checkout。

文章确认 Accel 领投 seed，Vercel Ventures、Okta Ventures、Anthropic 等参与；未披露金额。
