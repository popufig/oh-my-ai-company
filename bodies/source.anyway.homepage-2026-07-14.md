# Anyway 官网：Financial OS for Agents

URL: https://anyway.sh/

官网将 Anyway 定义为 “The Financial OS for Agents”，列出 Agent-native payments、x402/ACP/MPP 协议路由、secure sandbox、verifiable agent traces 和 SuperAPI。

首页还展示 Agent-to-Agent settlement、Agent 采购/广告支出、Agent service monetization 与 API 按次调用等场景。它能证明当前产品叙事，不能单独证明所有协议和功能已经生产可用。
