# Introducing nano 1.5
官方 2026-05-21 发布 nano 1.5，称其为第一版完整由上一版 production traces 驱动的 harness：每次 run、failure、recovery 都进入下一版改进数据。

供应商自报对比 nano-1：mean cost per cost-bearing task $2.33 → $1.56（-33%），p50 $1.49 → $1.21；mid-task auto-recovery 60% → 86%；median task time 9.5 → 7.6 min。文章还称归纳出 9 个 skills 处理生产失败。

这是产品工程方向的一手证据，不是独立评测。文章未披露样本量、任务分布、accepted outcome 或长期 retention。
