# Viktor product page

采集时间：2026-07-08。来源：browser read，正文可用。

产品页给出了 Viktor 的核心能力面：

- Add Viktor to Slack or Microsoft Teams。
- Connect tools：CRM、analytics、ad platforms、project management、version control 等，通过 OAuth。
- Data：从 Stripe、PostHog、Google Ads、HubSpot 等拉数据，产出 reports、charts、PDF、Excel。
- Automation：观察团队工作方式，建议 automations，例如 morning revenue briefings、weekly ad audits、anomaly alerts。
- Scheduling：Recurring workflows。
- Research：搜索 web、浏览真实网站、读文档、cross-reference sources。
- Apps：生成 dashboards、internal tools、calculators，含 databases、auth、hosting。
- Engineering：读 codebase、建 branches、写 code、开 PR、跑 tests。
- Browser automation：操作真实浏览器，填表、导航复杂流程、抓数据、截图。

产品页还直接把 Viktor 和 OpenClaw 做区分：Viktor 是 hosted/team-first/Slack+Teams/zero-config/3,200+ integrations，而 OpenClaw 被描述为 personal productivity/self-hosted/developer setup。
