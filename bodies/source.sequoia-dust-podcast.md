# Sequoia podcast: Dust cofounders on multiple custom agents

Sequoia Training Data podcast 采访 Dust co-founders [[person.gabriel-hubert]] 和 [[person.stanislas-polu]]。

关键点：

- Dust 创立于 2023 年初；两位 founder 有 Stripe / OpenAI 背景。
- Dust 的核心假设之一是 “one model will not rule them all”，企业需要 multi-model integration 和可切换模型层。
- 企业 AI 价值来自 proprietary data / data silos、security、integration 和 adoption。
- Dust 更倾向 augment human capabilities，而不是把 AI 作为简单替代劳动力。

研究价值：解释了 Dust 为什么不是单模型 app，而是做 agent/model/data/workflow 层。

关联公司：[[company.dust]]。
