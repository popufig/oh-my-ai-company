# Tasklet agent browser

URL: https://tasklet.ai/release-notes/2026-04-27-agent-browser
Legacy local source: `vertical/media/knowledge/radar/tasklet/sources/release-notes.cn.md`.

Evidence level: S1 official release note. Quality: full via legacy capture.

Key points:
- Every Tasklet agent gets a built-in Chrome browser for websites without APIs.
- Browser is faster/cheaper than full Computer Use VM for many web tasks.
- Login flow: user handles authentication in a live browser preview; agent resumes afterward.
- Persistent profile means login once, stay logged in for future tasks.
- Use cases include Google Ads screenshots, expense portals, QuickBooks reconciliation, Shopify orders and WordPress publishing.
