# AIsa raises $6.5M to build the AI Agent Resource Network

Key facts:
- AIsa announced $6.5M total funding including a new seed round co-led by Alibaba and Tribe Capital.
- Participants include Draper Associates, Sumitomo Corporation, Saison Capital and others.
- Funds intended for engineering, payment infrastructure, onboarding model/data/API providers, and stablecoin settlement.
- Product described as a unified resource and transaction layer for agents and developers to discover, access, and pay for resources such as AI models, APIs, real-time data, search services, SaaS tools, compute, and agent services.
- Mentions usage-based billing and fiat/stablecoin settlement.
- Customer example: Impossible Finance.
- Official traction claims: Feb-June 2026 registered agent users grew 150x; aggregate API calls/transactions grew 200x; over 50,000 registered agents without paid marketing.
- Claims AIsa ranked as top seller/server on the public x402 leaderboard and integrates with Circle, Visa, Stripe agent-payment initiatives.
