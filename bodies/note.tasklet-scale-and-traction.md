# Tasklet 规模与收入口径

结论：Tasklet 的规模信号强，但口径很多，必须拆开看。

可确认口径：
- 官方 YC launch / 官方融资公告：2026 年收入增长 >1,200%，达到或超过 $5M annualized run rate / ARR，$20M 融资，$175M 估值。证据：[[source.yc.tasklet]], [[source.release.tasklet-2026-04-07-funding]]。
- YC Jobs：融资写 $29.5M，团队 9 人。证据：[[source.yc-jobs.tasklet]]。
- Sacra：估计 2026 年 5 月达到 $10M ARR，2025 年底约 $385K。证据：[[source.sacra.tasklet]]。
- 社交传播/旧材料：有 $7M run-rate 说法，但发帖人/转述链存在口径风险。证据暂放 [[source.x.tasklet-archive-2026-06]] 和 [[source.community.tasklet-aggregate]]。

处理原则：
- 报告中不要写单一确定 ARR，除非指定口径。
- 推荐写法：官方披露至少 $5M run-rate/ARR、$20M round、$175M valuation；第三方 Sacra 估计 $10M ARR；社交传播另有 $7M run-rate 说法。
- `run rate` 不是全年已实现收入，社区里也有人质疑这个指标。
