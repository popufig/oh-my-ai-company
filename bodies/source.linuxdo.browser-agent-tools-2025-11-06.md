# linux.do: 智能自动操作浏览器项目讨论

证据等级：S3（中文开发者社区讨论）

这条讨论把 comet、browser-use、Stagehand、Browserbase 等放在“新一代浏览器智能 agent”集合里比较。用户需求包括图像识别、实时总结、自动化抓取感兴趣内容。

可用信号：
- Browserbase/Stagehand 在中文社区已被放入“智能操作浏览器”候选集合。
- 社区替代项包括 browser-use、Chrome DevTools MCP、本地 Chrome、传统 Playwright/Selenium 等。
- 有用户强调结构化快照比只返回图片更适合非视觉模型。
- 地域限制、本地浏览器启动失败、模型适配是使用门槛。

判断：Browserbase 的竞争不是单一公司，而是一组“本地浏览器 + MCP / 开源 browser-use / 云浏览器平台 / AI 浏览器”的能力组合。
