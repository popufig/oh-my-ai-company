# Sintra Workspace credits

官方 credits 文档给出任务消耗区间：普通聊天约 0.1–1，网站抓取约 0.5，基础 scheduled task 约 1–5，research 约 3–10，多工具复杂任务约 5–20+。workspace credits 用尽后 Helpers 停止工作；不同 workspace 之间不能共享余额。

Top-up 从 37.5 美元/150 credits 到 1200 美元/4800 credits，按月续费且不可退款。

**可支持：** Sintra 的真实计量单位是任务和模型消耗，不只是“12 个员工”的席位价格。250 credits 理论上约对应 12–50 次 5–20 credits 的复杂任务。

**边界：** 实际消耗取决于任务和失败重试，不能据此推算用户成本、毛利或使用频次。
