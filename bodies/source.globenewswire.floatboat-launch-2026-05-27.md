# Floatboat 全球发布 proactive agent OS

Floatboat 于 2026-05-27 通过 GlobeNewswire 发布官方 PR，宣布面向全球推出以日历为入口的 proactive agent OS，桌面客户端支持 macOS 与 Windows。稿件称 FloatSchedule 是首个核心产品，后续将扩展项目执行、客户管理和团队协调 Agent；连接 3,500+ 应用并保留 human-in-the-loop 审批。[[company.floatboat]] [[concept.calendar-as-agent-runtime]]

稿件由 Floatboat 自己发布，属于 S1 产品/发布证据，但其中 `San Francisco-based`、平均知识工作者每日切换应用 1,000 次、能力完备性等均带 PR 属性，不能视为独立验证。
