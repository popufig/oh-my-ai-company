# Minds docs overview

Source: https://hellominds.ai/docs/overview
Collected: 2026-07-09
Evidence: S1 official / quality full

Captured text says Animoca Minds is a service by [[company.animoca-brands]] and [[company.ethoswarm]] that lets anyone deploy and operate persistent AI agents called Minds without hardware or coding knowledge.

The docs define a Mind as an always-on digital employee that lives in the cloud, has its own identity, long-term memory, and on-chain wallet. It contrasts Minds with normal chatbots that reset after each session: a Mind can work autonomously 24/7, monitor data, execute multi-step tasks, and proactively reach out with results.

User interaction path: email and Telegram, natural language, email address enough to get started.
