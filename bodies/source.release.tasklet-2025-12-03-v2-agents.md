# Tasklet v2 agent architecture

URL: https://tasklet.ai/release-notes/2025-12-03-v2-agents
Legacy local source: `vertical/media/knowledge/radar/tasklet/sources/release-notes.cn.md`.

Evidence level: S1 official release note. Quality: full via legacy capture.

Key points:
- Rebuilt how Tasklet agents think, plan and execute.
- One agent can handle multiple/unlimited triggers instead of duplicating agents.
- Agents get task lists to break down complex work and queue subtasks.
- Agents can spawn specialized sub-agents for complex workflows.
- Context management was rebuilt to handle more data and long-running tasks.
