Fetch API 可以把网页加载到 cloud browser 中，并返回 markdown、HTML、links、screenshot、JSON、branding profile。JSON extraction 支持 prompt/schema，两者可结合；还支持 include/exclude selector、navigation waitUntil/waitFor/timeout、cache、stealth、browser profile、captcha、location。

这说明 Hyperbrowser 在向上吃网页读取、结构化抽取和 web data API，而不是停在 browser session。
