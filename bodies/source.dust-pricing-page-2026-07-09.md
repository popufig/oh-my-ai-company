# Dust pricing page

采集时间：2026-07-09。

Pricing page 标题：Dust Pricing: Business and Enterprise Plans for AI Agents。

价格与包装：

- Business: for teams up to 100 people。
- Free: $0，500 credits lifetime。
- Pro: $24 / seat / month yearly，8,000 credits / seat / month。
- Max: $120 / seat / month yearly，40,000 credits / seat / month。
- Enterprise: custom，面向 at scale AI。
- Programmatic usage rate: $0.01 / credit；Enterprise 可 custom。

Business 包含：20+ frontier models、custom agents with skills/knowledge/tools、multi-agent workflows on schedules and triggers、Slack/Notion/GitHub/Drive + 20 more 或 any tool via MCP、team collaboration workspaces、SSO、US/EU data residency。

Enterprise 增强：unlimited connectors & MCP servers、workspace-pooled credits、SCIM、audit logs、custom data retention、US data residency & single-tenant deployment、dedicated CSM、priority support & SLA、custom legal terms。

功能对比里出现关键概念：

- Multi-agent orchestration & triggers。
- MCP servers native + remote。
- Frames: interactive dashboards & apps。
- Pods: collaborative workspaces with shared context。
- Search/query/extract across all company data。
- Automation platforms: Zapier、Make、n8n、Power Automate。
- Developer API: Conversation API + Data Source API。

研究判断：Dust 定价是 seat + credits hybrid。它比 Viktor 的 pure credits 更接近传统 enterprise SaaS 包装，同时保留模型/agent 使用量计费。
