# Agent Trust Stack：身份、访问、授权与支付的组合层

Agent 商业基础设施不能只解决“怎么付钱”。在真实互联网中，Agent 还要穿过 Bot 管理、登录、身份验证、用户委托、风险决策和争议责任。Agent Trust Stack 是把这些环节组合成统一凭证和 policy enforcement 的产品模式。

[[company.skyfire]] 是当前样本：以 KYA 绑定人、Agent 和平台，将同一身份凭证用于访问、登录和 checkout，再借 Experian、Akamai、Fastly 等伙伴进入商户基础设施。证据：[[source.skyfire.home-2026-07-14]]、[[source.skyfire.security-doc-2026-07-14]]、[[source.experian.skyfire-agent-trust-2026-05-15]]、[[source.fastly.skyfire-partnership-2026-06-10]]。

待验证：这是否会形成独立品类，还是最终被卡组织、身份平台、CDN/WAF 或支付 PSP 吸收。当前置信度为 medium。
