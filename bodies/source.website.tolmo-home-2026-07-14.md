# Tolmo 官网与平台总览

采集时间：2026-07-14。官网定位为：在代码、云和供应商形成的 live production knowledge graph 上运行 autonomous security agents。

平台公开三类 Agent：Pentesting、Internal Discovery、Remediation。工作流为 Gather -> Connect -> Time travel；接入强调 read-only roles、无需部署 agent、无需修改代码。

官网还声称安全团队平均需 14 天响应、攻击者约 1.5 天可利用新漏洞。这些是营销引用，不作为 Tolmo 自身效果数据。
